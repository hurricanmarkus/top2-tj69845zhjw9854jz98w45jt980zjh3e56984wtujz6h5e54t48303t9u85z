# GESCHENKE IMPORT - MANUELLE ANLEITUNG

## Option 1: Über Firebase Console (EINFACHSTE METHODE)

1. Gehe zu: https://console.firebase.google.com/
2. Wähle dein Projekt: top2-app
3. Gehe zu: Firestore Database
4. Navigiere zu: artifacts/top2-tj69845zhjw9854jz98w45jt980zjh3e56984wtujz6h5e54t48303t9u85z/public/data/users/[DEINE_UID]
5. Erstelle manuell:
   - Collection: geschenke_themen
   - Für jedes Thema ein Dokument
   - Unter jedem Thema eine Subcollection: geschenke

## Option 2: CSV-Import in Excel/Google Sheets

Deine Daten sind bereits in Excel-Format. Behalte sie dort und:
1. Erstelle die Themen manuell in der App (6 Themen)
2. Kopiere die Geschenke pro Thema
3. Füge sie einzeln über die App ein
